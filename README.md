Dynamite Sports Learner is an interactive web platform designed to help youths confidently learn new sports. The platform addresses the common challenge of not knowing how to begin learning a sport by providing structured guidance, expert support, and clear learning paths.

## ğŸ¯ Project Brief

- **Project Name:** Dynamite Sports Learner
- **One-Line Pitch:** Learn new Sports, Get active, pick up new skills with our interactive and amazing website
- **Why It Matters:** Many youths are interested in learning new sports but often feel unsure about how to begin, where to find reliable guidance, or which skills to learn first. ### Prerequisites

### Installation

1. **Clone or download the project**
   ```bash

   cd DynamiteSportsLearner
   ```

2. **Open in VS Code**
   ```bash
   code .
   ```
